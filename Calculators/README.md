# Calculators

Two Python calculators — a GUI app and a terminal app.

## Smart Calculator (`smart_calculator.pyw`)

A premium GUI calculator built with Tkinter.

### Features

- **3 Themes** — Dark, Light, Neon Red (switch anytime)
- **Neon Glow Buttons** — animated hover, press, and ripple effects
- **Floating Particles** — background particle field with glowing orbs
- **Gradient Bars** — animated rainbow gradients
- **Scientific Functions** — sin, cos, tan, log, ln, sqrt, power, factorial
- **Angle Modes** — RAD / DEG toggle
- **Smart Parentheses** — auto-detects open/close
- **Keyboard Support** — full keyboard input
- **Smooth Animations** — font scaling, flash effects, fade-in

### Run

```bash
python smart_calculator.pyw
```

Or double-click the file on Windows (`.pyw` runs without console).

---

## Terminal Calculator (`calculator.py`)

A colorful terminal calculator with ASCII art UI.

### Features

- **9 Operations** — add, subtract, multiply, divide, power, sqrt, percentage, factorial, expression mode
- **Expression Mode** — type full math expressions with parentheses, functions (sin, cos, sqrt, log, pi, e)
- **Calculation History** — view all past calculations
- **Colored UI** — ANSI-colored menus and results

### Run

```bash
python calculator.py
```

### Controls

| Key | Action |
|-----|--------|
| 1-8 | Math operations |
| 9 | Expression mode |
| H | View history |
| C | Clear screen |
| Q | Quit |
