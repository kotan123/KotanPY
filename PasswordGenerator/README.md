# Password Generator

Secure password generator with a dark space-themed UI. Built with Python and CustomTkinter.

## Features

- **Adjustable length** — 4 to 64 characters via slider
- **Character types** — uppercase (A-Z), lowercase (a-z), digits (0-9), symbols (!@#)
- **Strength meter** — 4-segment visual bar (Weak / Fair / Strong / Very Strong)
- **Copy to clipboard** — one-click copy with visual feedback
- **Recent history** — last 5 generated passwords with Clear button
- **Custom background** — space-themed dark UI
