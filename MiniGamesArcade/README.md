# Mini Games Arcade

12 mini games in one app, built with Python & Pygame. No external assets needed — everything is drawn with code.

## Games

| # | Game | Controls |
|---|------|----------|
| 1 | Pong | W/S or Arrow Up/Down |
| 2 | Flappy Bird | SPACE |
| 3 | Breakout | A/D or Arrow Left/Right |
| 4 | Space Invaders | A/D + SPACE |
| 5 | Asteroids | A/D turn, W gas, SPACE shoot |
| 6 | Memory Match | Mouse click |
| 7 | Dodge Master | WASD or Arrows |
| 8 | Color Catch | A/D or Arrow Left/Right |
| 9 | Meteor Storm | WASD + SPACE |
| 10 | Platform Jump | A/D + SPACE (double jump!) |
| 11 | Tank Battle | WASD + Mouse aim + Click shoot |
| 12 | Rhythm Tap | D F J K |

## Features

- 4 color themes: Neon, Ocean, Sunset, Matrix (switch in main menu)
- Particle effects, glow, animations
- Control hints at the start of each game
- No images or fonts required — pure code graphics
- ESC to return to menu from any game

## Download & Play

Go to [Releases](../../releases) and download `MiniGamesArcade.exe` — just run it, no Python needed!

## Run from Source

```bash
pip install pygame
python mini_games.pyw
```
