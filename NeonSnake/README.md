# Neon Snake

A modern Snake game with neon aesthetics, built with Python and Pygame.

## Features

- **10 Color Themes** — Neon, Cyberpunk, Ocean, Forest, Retro, Void, Sunset, Arctic, Lavender, Blood
- **3 Game Modes**
  - Classic — no walls, snake wraps around edges
  - Walls — deadly borders + level obstacles that appear as you progress
  - Speed — starts fast and gets faster with every food eaten
- **Smooth Movement** — interpolated snake movement with easing functions
- **Particle System** — explosions, fireworks, confetti, spirals, sparks
- **Combo System** — chain food pickups for score multipliers
- **Powerups** — shield, rainbow mode, shrink
- **Bonus Items** — star pickups for extra points
- **Animated Menu** — smooth selection transitions, glowing highlights, floating particles
- **Score Saving** — local high scores for each game mode
- **Sound Effects** — procedurally generated retro sounds

## Requirements

```bash
pip install pygame
```

## How to Play

```bash
python snake.pyw
```

Or double-click `run.bat` on Windows.

### Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move snake |
| Enter | Start game / Restart |
| T | Change theme |
| P / Space | Pause |
| Esc | Back to menu |

## Themes

Switch between 10 themes anytime by pressing **T**:

Neon, Cyberpunk, Ocean, Forest, Retro, Void, Sunset, Arctic, Lavender, Blood

Each theme changes the entire color palette — background, snake, food, UI, particles, and effects.
